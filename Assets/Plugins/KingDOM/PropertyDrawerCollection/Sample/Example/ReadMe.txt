Editor’s design is implemented entirely through PropertyDrawer.
AssertSample.unity - Script "AssertSetup.cs" is used when checking in the plugin Unity Test Tools of Asset Store.
AssertExtSample.unity - Extended version AssertSample. Script "AssertSetupExt.cs" can be used instead of "AssertSetup.cs".
Editor/WindowUsePropertyDrawer.cs - Using PropertyDrawer when creating EditorWindow. To see an example, uncomment the line with the MenuItem and activate the menu "KingDOM / Window VS PropertyDrawer".
